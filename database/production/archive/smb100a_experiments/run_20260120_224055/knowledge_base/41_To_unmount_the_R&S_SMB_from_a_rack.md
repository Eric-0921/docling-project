# To_unmount_the_R&S_SMB_from_a_rack

To unmount the R&S SMB from a rack

Loosen the screws at the rack brackets.

Remove the R&S SMB from the rack.

If placing the R&S SMB on a bench top again, unmount the adapter kit from the R&S SMB. Follow the instructions provided with the adapter kit.
