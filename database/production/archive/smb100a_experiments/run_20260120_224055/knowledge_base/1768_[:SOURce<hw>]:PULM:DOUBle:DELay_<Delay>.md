# [:SOURce<hw>]:PULM:DOUBle:DELay_<Delay>

[:SOURce<hw>]:PULM:DOUBle:DELay <Delay>

Sets the delay from the start of the first pulse to the start of the second pulse.

Parameters:

<Delay>

float

Range:

10 ns  to  100 s

Increment:

5 ns

*RST:

3 us

Example:

PULM:DOUB:DEL 22 us

22 us elapse between the beginning of the first pulse and the beginning of the second pulse in double-pulse mode.

Options:

R&S SMB-K23 (Pulse Generator)

Manual operation:

See "Double Pulse Delay - Pulse Generator" on page 233
