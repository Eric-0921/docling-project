# Return_values:

Return values:

<DEVice>

0 | 1 | OFF | ON

Example:

SENS:STAT?

queries if a sensor is connected to the instrument.

Response: 1

a sensor is connected to the POWER SENSOR interface.

Usage:

Query only

Manual operation:

See "Sensor" on page 172
