# [:SOURce<hw>]:POWer:SPC:CRANge_<PowCntrlCRange>

[:SOURce<hw>]:POWer:SPC:CRANge <PowCntrlCRange>

Defines the capture range of the power control system.

Within the range:

Target Level +/- Catch Range the power control locks and tries to achieve the target level. Readings outside the range are not considered.
