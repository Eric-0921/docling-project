# Parameters:

Parameters:

<Phase>

float

Range:

-720  to  720

Increment:

0.1

*RST:

0

Example:

PHAS 0.1 RAD

changes the phase by 0.1 RAD relative to the current phase. PHAS:REF

adopts the set phase as the current phase.

Manual operation:

See "Delta Phase" on page 142
