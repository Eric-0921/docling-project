# Return_values:

Return values:

<Meas>

0 | 1

Example:

CAL:STER:ANAL?

starts the adjustments for analog channels of the stereo coder.

Response: 0

the adjustments have been performed successfully.

Usage:

Query only

Options:

R&S SMB-B5

Manual operation:

See "Adjust Stereo Coder" on page 496
