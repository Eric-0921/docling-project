# Disabling_Remote_Access_via_Ultr@VNC

Disabling Remote Access via Ultr@VNC

The VNC connection can be disabled by removing the program on the instrument or by deactivating the VNC Server service.

Remove the VNC program

Press the "Windows" key to access the operating system and open the "Add or Remove Programs" by selecting "Start > Settings > Control Panel > Add or Remove Programs".

Remove the VNC program.

Deactivate the VNC Server service

Press the "Windows" key to access the operating system and open the "Services" by selecting "Start > Settings > Control Panel > Services".

Deactivate the VNC Server service.

The connection is disabled, the VNC icon disappears from the task bar of the instrument.
