# Varying_a_value

Varying a value

Activate the editing mode.

Set the cursor to the left of the number to be changed using the [Left/Right] arrow keys.

The value at the cursor position is varied.

To vary the selected value, use the [Up/Down] arrow key or turn the rotary knob. The value is increased or decreased.
