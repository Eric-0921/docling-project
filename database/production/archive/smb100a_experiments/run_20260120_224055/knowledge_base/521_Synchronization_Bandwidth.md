# Synchronization_Bandwidth

Synchronization Bandwidth

Selects the synchronization bandwidth for an external reference signal.

"Narrow"

Synchronization bandwidth is 50 Hz.

"Wide"

The synchronization bandwidth is approximately 350 Hz. This mode is useful for very precise reference sources of high spec- tral purity.

Remote command:

[:SOURce]:ROSCillator:EXTernal:SBANdwidth on page 411
