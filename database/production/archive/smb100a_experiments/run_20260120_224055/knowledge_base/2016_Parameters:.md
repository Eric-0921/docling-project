# Parameters:

Parameters:

<State>

0 | 1 | OFF | ON

*RST:

n.a. (factory preset: 0)

Example:

SYST:KLOC ON

Locks the front panel and external controls. To enable the con- trols, set SYST:KLOC OFF .
