# Accessing_files_with_user_data

Accessing files with user data

To access an editable user data file, select the "Save/Recall" or "File Manger" function in the individual dialog.

To access a loadable data file, select the "Select/New" or "File Manager" function in the individual dialog.

To access the "File Manager" function, press the [SETUP] (CTRL+E) key and select "Save/Recall > File Manager".

A "File Select" window for loading, saving or creating a file or the "File Manager" dialog for managing all files is displayed.
