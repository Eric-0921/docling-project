# STEReo:DIRect_"PRESET"

STEReo:DIRect "PRESET"

Sets the default settings in accordance with specifications.

Example:

STEReo:DIRect "PRESET"

Usage:

Event

STEReo:DIRect "PS=<PS>"

STEReo:DIRect? "PS"

Sets or reads the RDS program service (PS) name.
