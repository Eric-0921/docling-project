# Editing_a_value_in_the_overwrite_mode

Editing a value in the overwrite mode

Activate the editing mode.

Set the cursor on the numeric digit to be changed using the [Left/Right] arrow keys. The cursor is displayed in blue and the number to be replaced is highlighted.

Click on a numeric key to overwrite the highlighted value.
