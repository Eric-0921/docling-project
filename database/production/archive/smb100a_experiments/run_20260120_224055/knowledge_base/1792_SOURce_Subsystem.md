# SOURce_Subsystem

SOURce Subsystem

Example:

PULM:SOUR INT

selects the internal modulation source.

PULM:STAT ON

activates the pulse modulation.

Usage:

SCPI confirmed

Options:

R&S SMB-K21 or R&S SMB-K22 (Pulse Modulator)

Manual operation:

See "Source" on page 216
