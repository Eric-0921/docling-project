# Parameters:

Parameters:

<State>

0 | 1 | OFF | ON

*RST:

OFF

Example:

SENS1:POW:OFFS 0.4dB

sets a level offset of 0.4 dB

SENS1:POW:OFFS:STAT ON

a level offset of 0.4 dB is added to the measured value.

Manual operation:

See "Level Offset" on page 175
