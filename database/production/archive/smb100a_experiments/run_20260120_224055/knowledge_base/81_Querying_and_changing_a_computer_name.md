# Querying_and_changing_a_computer_name

Querying and changing a computer name

Press the "Setup" key and select "Network Settings". The computer name is displayed under "Hostname".

Press the "Setup" key, select "Protection" and enable the "Protection Level 1". The default password is 123456 .

The parameter "Hostname" in the "Network Settings" dialog is now enabled for configuration.

Change the "Hostname".
