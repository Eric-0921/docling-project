# Parameters:

Parameters:

<State>

0 | 1 | OFF | ON

*RST:

0

Example:

OUTP:FILT:AUTO 0

OUTP:FILT:LPAS:STAT 1

Selects manual mode and activates the low harmonic filter.

OUTP:FILT:STAT?

Queries the current filter activity.

Manual operation:

See "State" on page 151
