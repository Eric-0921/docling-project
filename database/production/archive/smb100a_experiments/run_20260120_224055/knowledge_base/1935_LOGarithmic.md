# LOGarithmic

LOGarithmic

With the logarithmic sweep, the step width is a constant fraction of the current frequency. This fraction is added to the current frequency. The logarithmic step width is entered in % (see [:

SOURce<hw>]:SWEep[:FREQuency]:STEP:LOGarithmic on page 428).

*RST:

LINear

Example:

SWE:SPAC LIN

selects linear sweep spacing for a frequency sweep at the RF output.
