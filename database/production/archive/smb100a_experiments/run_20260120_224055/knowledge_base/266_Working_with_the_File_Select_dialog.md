# Working_with_the_File_Select_dialog

Working with the File Select dialog

Access the "File Select" dialog (see "Accessing files with user data" on page 87).

Navigate in the "File Select" dialog.

Load an existing file.

In the "Recall Settings" dialog, mark a file and press the "Select" button.

Save a file. In the "Save Settings" dialog, enter file name in the "File Name:" field. Select the directory to which the file is to be saved and then select the "Save" button.

5. Create a new file. To create a new file, use the "Save Settings" functionality, i.e. specify file name and directory and save the file. The created file is empty; it must be filled with the necessary values in the individual editor.
