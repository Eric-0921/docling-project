# Parameters:

Parameters:

<Stop>

float

Range:

full frequency range

Increment:

see the data sheet: RF characteristics > Resolution of setting 500 MHz

*RST:

Example:

FREQ:STOP 2 GHz

sets the stop frequency for the frequency sweep to 2 GHz.

FREQ:STAR 1 MHz

sets the start frequency for the frequency sweep to 1 MHz.

Manual operation:

See "Stop Freq - Frequency Sweep" on page 184
