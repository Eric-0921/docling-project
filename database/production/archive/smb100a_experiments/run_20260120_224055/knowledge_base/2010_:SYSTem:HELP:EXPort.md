# :SYSTem:HELP:EXPort

:SYSTem:HELP:EXPort

Saves the online help as zip archive in the user directory.

Saves the online temp as zip archive if usee directory.
                Example:                :SYSTEM:HELPE:EXPORT
                                    MMEM:CDIR?
                                    // "/var/user"
                                    MMEM:CAT?
                                    //
