# Setting_parameters:

Setting parameters:

<Filename>

Example:

MMEM:CDIR '/var/user/Lists'

selects the directory for the pulse train files.

PULM:TRA:DEL 'P_FIVE'

deletes the list P_FIVE

Usage:

Setting only

Options:

R&S SMB-K27 (Pulse Train)

Manual operation:

See "Pulse Train Data - Pulse Generator" on page 236
