# Return_values:

Return values:

<Pocount>

float

Example:

DIAG:INFO:POC

queries the number of power on events.

Response: 123

The instrument was switched on for 123 times up to now.

Usage:

Query only

Manual operation:

See "Power On Count" on page 98
