# Setting_parameters:

Setting parameters:

<RetrNumb>

Number of retransmissions

<DataSeq>

Max. 20 different data sequences can be defined. 10 characters must be specified each per <DataSeq>. Leading zeros, if any, must also be specified. 00: erases the data.

Range:

0000000000  to  1FFFFFFFFF (10 ASCII ccoded hexadecimal characters each)
