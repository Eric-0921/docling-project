# Query_parameters:

Query parameters:

<Force>

string

*RST:

force

Return values:

<Measure>

0 | 1

Example:

CAL:LEV:MEAS?

starts adjustments for maximum level accuracy.

Response:

"0"

adjustment has been performed successfully.

Usage:

Query only

Manual operation:

See "Adjust Level" on page 495
