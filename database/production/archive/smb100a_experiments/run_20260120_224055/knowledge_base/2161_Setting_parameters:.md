# Setting_parameters:

Setting parameters:

<Hour>:<Min>:<Sec> Range:

00:00:00  to  23:59:59

<Day>.<Month>.<Year> Range:

01.01.00  to  31.12.85

Example:

STEReo:DIRect "CT=20:30:59,01.08.03"

The real-time clock is set to 20:30:59 and 1 August 2003.

Example:

STEReo:DIRect? "CT"

Response:

"20:31:06,01.08.03"
