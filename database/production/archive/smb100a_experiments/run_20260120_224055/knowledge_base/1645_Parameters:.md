# Parameters:

Parameters:

<Index>

integer

*RST:

0

Example:

LIST:SEL '/var/list3'

selects list3 for use in List mode.

FREQ:MODE LIST

activates List mode. List3 is processed.

LIST:MODE STEP

selects manual, step-by-step processing of the list.

LIST:IND 5

the frequency/level value pair with index 5 is executed.

TRIG:LIST:SOUR SING

selects triggering by means of the single trigger. The list is exe- cuted once.

SOUR:LIST:TRIG:EXEC

triggers the processing of the selected list.

See "Current Index - List Mode" on page 196
