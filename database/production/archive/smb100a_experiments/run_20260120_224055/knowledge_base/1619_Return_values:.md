# Return_values:

Return values:

<Catalog>

string

Example:

MMEM:CDIR '/var/Listmode'

selects the directory for the list mode files. LIST:CAT?

queries the available lists.

Response: 'list1,list2' the lists list1 and list2 are available.

Usage:

Query only
