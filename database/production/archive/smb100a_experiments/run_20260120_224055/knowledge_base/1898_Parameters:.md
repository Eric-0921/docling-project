# Parameters:

Parameters:

<State>

0 | 1 | OFF | ON

*RST:

OFF

Example:

STER:RDS:TRAF:PROG:STAT ON

activates RDS traffic program.

STER:RDS:TRAF:ANN:STAT ON

activates RDS traffic announcement.

Options:

R&S SMB-B5

Manual operation:

See "RDS Traffic Announcement State - Stereo Modulation" on page 223
