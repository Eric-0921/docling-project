# Parameters:

Parameters:

<Frequency>

5MHZ | 10MHZ

*RST:

n.a. (factory preset: 10MHZ)

Example:

ROSC:SOUR EXT

Selects the external source. The reference must be input at the REF IN input.

ROSC:EXT:FREQ 10MHz

Selects 10 MHz external reference frequency.

Manual operation:

See "External Reference Frequency" on page 145
