# Setting_parameters:

Setting parameters:

<PTYN>

8 ASCII characters

Note: An eight-digit value must always be set. Blank spaces, if any, must also be entered, otherwise the value will not be accep- ted.

Example:

STEReo:DIRect "PTYN=Football"

Sets the program type name to be transmitted to "Football". STEReo:DIRect "GS=0A,10A'

Group 10A is activated in addition to group 0A. The program type name "Football" is now transmitted.
