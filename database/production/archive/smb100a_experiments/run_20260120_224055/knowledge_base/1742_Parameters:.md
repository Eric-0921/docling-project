# Parameters:

Parameters:

<Power>

Minimum level

The value range for the level setting varies according to the instrument model.

The values are given in the data sheet.

Increment:

0.01

*RST:

-30

Example:

SOUR:POW:POW 15

sets the RF level at output to 15 dBm.

Manual operation:

See "Amplitude" on page 149
