# Parameters:

Parameters:

<State>

0 | 1 | OFF | ON

*RST:

0

Example:

TRIG0:SWE:SOUR SING

FREQ:MODE SWE

SWE:SHAP SAWT

SWE:RETR ON

activates retrace function, i.e. the frequency changes to the value at start frequency while waiting for the next trigger event.

Manual operation:

See "Retrace - RF Frequency Sweep" on page 185
