# [:SOURce<hw>]:SWEep:POWer:EXECute

[:SOURce<hw>]:SWEep:POWer:EXECute

Triggers a sweep.

The command is only valid for sweep mode Single ( SOURce:SWEep:POWer:MODE SINGle ). The command corresponds to the manual-control command "Execute Single Sweep".

Example:

SOURce:SWEep:POWer:MODE SINGle

sets the single cycle mode of the level sweep.

SWE:POW:EXEC

triggers a level sweep at the RF output.

Usage:

Event

Manual operation:

See "Execute Single Sweep - Level Sweep" on page 190
