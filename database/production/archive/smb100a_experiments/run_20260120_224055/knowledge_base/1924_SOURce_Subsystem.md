# SOURce_Subsystem

SOURce Subsystem

Example:

FREQ:STAR

sets the start frequency to 100 MHz.

FREQ:STOP

sets the stop frequency to 500 MHz.

SWE:SPAC LIN

sets linear sweep spacing.

SWE:POIN 401

sets 401 sweep steps for linear sweep spacing. The sweep step width ( STEP ) is automatically set to 1 MHz.
