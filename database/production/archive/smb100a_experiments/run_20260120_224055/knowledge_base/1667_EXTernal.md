# EXTernal

EXTernal

The list is triggered externally via the [INST TRIG] connector. The list is executed once.

*RST:

AUTO

Example:

LIST:TRIG:SOUR EXT

selects triggering by means of the external trigger.

Manual operation:

See "Mode - List Mode" on page 194
