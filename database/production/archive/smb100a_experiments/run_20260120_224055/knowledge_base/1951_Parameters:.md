# Parameters:

Parameters:

<State>

0 | 1 | OFF | ON

*RST:

0

Example:

TRIG0:SWE:SOUR SING

POW:MODE SWE

SWE:POW:SHAP SAWT

SWE:POW:RETR ON

activates retrace function, i.e. the level changes to the value at start level while waiting for the next trigger event.

Manual operation:

See "Retrace - RF Level Sweep" on page 191
