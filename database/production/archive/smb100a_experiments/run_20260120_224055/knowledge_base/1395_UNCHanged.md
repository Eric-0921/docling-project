# UNCHanged

UNCHanged

restores the initial state of the RF output before the last turn off. sets the output status as it was when the instrument was switched off.

*RST:

n.a. (factory preset: UNCHanged)

Example:

OUTP:PON OFF

RF output A is deactivated when the instrument is switched on.

Manual operation:

See "Power-On State - RF Signal" on page 153
