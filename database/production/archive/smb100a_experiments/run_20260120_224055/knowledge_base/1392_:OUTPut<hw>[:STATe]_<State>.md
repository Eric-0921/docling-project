# :OUTPut<hw>[:STATe]_<State>

:OUTPut<hw>[:STATe] <State>

Activates and deactivates the RF output signal (RF ON / RF OFF).

Parameters:

<State>

Example:

0 | 1 | OFF | ON

*RST: 0

OUTP OFF

deactivates the RF output.

Manual operation:

See "RF On" on page 137
