# :SYSTem:COMMunicate:NETWork:MACaddress_<MacAddress

:SYSTem:COMMunicate:NETWork:MACaddress <MacAddress>

Queries the MAC address of the network adapter.

Parameters:

<MacAddress>

string

Example:

SYST:COMM:NETW:MAC

queries the MAC address.
