# [:SOURce<hw>]:PHASe:REFerence

[:SOURce<hw>]:PHASe:REFerence

Adopts the phase set with SOURce:PHASe:ADJust as the current phase.

Example:

PHAS 0.1RAD

changes the phase by 0.1 RAD relative to the current phase. PHAS:REF

adopts the set phase as the current phase.

Usage:

Event

Manual operation:

See "Reset Delta Phase Display" on page 142
