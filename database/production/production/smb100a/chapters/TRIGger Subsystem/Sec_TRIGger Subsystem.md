# TRIGger Subsystem
Example:

SWE:POW:MODE AUTO

selects the triggered sweep mode, i.e. a trigger is required to start the sweep.

TRIG:PSW:SOUR SING

sets the single trigger mode, i.e. a trigger starts a single sweep. TRIG:PSW

starts a single RF level sweep.
