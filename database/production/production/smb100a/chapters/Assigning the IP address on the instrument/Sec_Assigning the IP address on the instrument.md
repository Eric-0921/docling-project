# Assigning the IP address on the instrument
Press the [Setup] key and select the "Network Settings" dialog.

Set the "Address Mode" to Static.

Select the "IP Address" and enter the IP address, for example 192.168.0.1. . The IP address consists of four number blocks separated by dots. Every block contains 3 numbers in maximum.

Select the "Subnet Mask" and enter the subnet mask, for example 255.255.255.0 . The subnet mask consists of four number blocks separated by dots. Every block contains 3 numbers in maximum.
