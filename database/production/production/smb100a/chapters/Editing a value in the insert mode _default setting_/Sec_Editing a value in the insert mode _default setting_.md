# Editing a value in the insert mode (default setting)
Press the rotary knob (= Enter) to activate the editing mode.

Getting Started

Instrument Control

If the cursor is placed at the right of the total value, the insert mode is always active.

Set the cursor to the left of the number to be changed using the [Left/Right] arrow keys.

The cursor is displayed in green.

Click on a numeric key to insert a new value.
