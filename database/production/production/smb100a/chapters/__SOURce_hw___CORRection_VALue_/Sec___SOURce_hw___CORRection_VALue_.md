# [:SOURce<hw>]:CORRection:VALue?
Queries the current value for user correction.

Return values:

<Value>

float

Range:

-100  to  100

Increment:

0.01

*RST:

0

Example:

CORR:VAL?

queries the value currently used for level correction.

Response:

-3

the correction value is - 3 dB.

Usage:

Query only

Manual operation:

See "User Correction Value - User Correction" on page 160
