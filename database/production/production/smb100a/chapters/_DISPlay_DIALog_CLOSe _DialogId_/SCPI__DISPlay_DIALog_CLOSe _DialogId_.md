# :DISPlay:DIALog:CLOSe <DialogId>
Closes the specified dialog. To determine the dialog identifier, use command : DISPlay:DIALog:ID? .


## Setting parameters:
<DialogId>

string

Example:

DISP:DIAL:CLOS "<dialog ID>"

Closes the dialog, determined with the "<dialog ID>".

Usage:

Setting only
