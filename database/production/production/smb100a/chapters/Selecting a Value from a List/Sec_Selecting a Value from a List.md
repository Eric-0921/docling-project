# Selecting a Value from a List
Selection lists provide a list of predefined values for the selected parameter.

To select a item from a list, proceed as follow:

Press [ENTER] key to open the list.

Use one of the following alternatives to navigate through the list:

Turn the rotary knob or use the [Up/Down] arrow keys. The selected item is highlighted.

Press [TOGGLE ON/OFF] key several times until the preferred entry is displayed in the selection field.

To confirm the selection, press the [ENTER] key.
