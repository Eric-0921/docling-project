# :DISPlay:DIALog:ID?
Returns the dialog identifiers of the open dialogs in a string separated by blanks.


## Return values:
<DialogIdList>

string

Example:

DISP:DIAL:ID?

Response: "<dialog ID(1)> <dialog ID(2)> ...

<dialog ID(n)>"

Returns the dialog identifiers of all opened dialogs.

Usage:

Query only
