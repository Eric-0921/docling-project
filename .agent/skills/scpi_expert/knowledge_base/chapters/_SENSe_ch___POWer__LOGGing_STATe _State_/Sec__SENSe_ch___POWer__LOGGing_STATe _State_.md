# :SENSe<ch>[:POWer]:LOGGing:STATe <State>
Activates the recording of the power values, measured by a connected R&S NRP power sensor.

Parameters:

<State>

0 | 1 | OFF | ON

*RST:

0

Example:

SENS:LOGG:STAT ON

activates recording of the power measurement of the first sen- sor.

Manual operation:

See "Enable Logging" on page 177
