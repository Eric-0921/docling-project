# Starting a Remote Control via the LXI web browser
This section assumes that the instrument and the controller PC are connected in the LAN.

Start a web browser that supports html5 (W3C compliant).

Enter the IP address of the R&S SMB in the browser's address bar. The R&S SMB's welcome page is displayed.
