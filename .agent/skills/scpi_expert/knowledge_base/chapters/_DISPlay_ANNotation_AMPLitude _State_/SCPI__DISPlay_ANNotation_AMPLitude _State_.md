# :DISPlay:ANNotation:AMPLitude <State>
Indicates asterisks instead of the level values in the status bar.


## Parameters:
<State>

0 | 1 | OFF | ON

*RST:

n.a. (factory preset: 1)

Example:

DISP:ANN:AMPL ON

Suppresses the level display.

Manual operation:

See "Annotation Amplitude" on page 119
