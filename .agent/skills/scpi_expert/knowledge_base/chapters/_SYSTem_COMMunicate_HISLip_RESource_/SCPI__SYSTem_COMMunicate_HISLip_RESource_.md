# :SYSTem:COMMunicate:HISLip:RESource?
Queries the VISA resource string, used for remote control of the instrument with HiSLIP protocol.


## Return values:
<Resource>

string

Example:

SYSTem:COMMunicate:HISLip:RESource?

Response: "TCPIP::192.1.2.3::hislip0::INSTR"

Usage:

Query only

Manual operation:

See "VISA Resource Strings" on page 110
