# :SYSTem:COMMunicate:USB:RESource?
Queries the VISA resource string for remote control via the USB interface.


## Return values:
<Resource>

string

Example:

SYSTem:COMMunicate:USB:RESource?

queries the VISA resource string for remote control via the USB interface.

Response: "USB::72::000000::INSTR"

Usage:

Query only

Manual operation:

See "VISA Resource Strings" on page 110
