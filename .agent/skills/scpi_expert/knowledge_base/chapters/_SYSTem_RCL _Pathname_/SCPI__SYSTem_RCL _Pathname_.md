# :SYSTem:RCL <Pathname>
Loads a file with previously saved R&S SMB settings.

Loads the selected file with previously saved R&S SMB settings from the default or the specified directory. Loaded are files with extension *.savrcltxt .


## Setting parameters:
<Pathname>

Example:

SYSTem:RCL "/var/user/temp/Test" Loads the file Test.savrcltxt from the direc- tory /var/user/temp/ .

Usage:

Setting only

Manual operation:

See "Recall" on page 131
