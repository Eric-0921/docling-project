# :DISPlay:ANNotation:FREQuency <State>
Indicates asterisks instead of the frequency values in the status bar.


## Parameters:
<State>

0 | 1 | OFF | ON

*RST:

n.a. (factory preset: 1)

Example:

DISP:ANN:FREQ ON

Supresses the frequency display.

Manual operation:

See "Annotation Frequency" on page 119
