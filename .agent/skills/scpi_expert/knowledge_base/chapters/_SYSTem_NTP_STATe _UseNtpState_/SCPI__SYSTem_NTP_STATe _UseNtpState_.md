# :SYSTem:NTP:STATe <UseNtpState>
Activates clock synchronization via NTP.

Parameters:

<UseNtpState>

0 | 1 | OFF | ON

*RST:

n.a. (factory preset: 0)

Example:

SYSTem:NTP:STATe 1

Manual operation:

See "Use Time from NTP Server" on page 492
