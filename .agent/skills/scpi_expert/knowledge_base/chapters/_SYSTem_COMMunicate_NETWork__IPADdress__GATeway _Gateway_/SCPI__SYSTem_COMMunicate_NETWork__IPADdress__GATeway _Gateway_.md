# :SYSTem:COMMunicate:NETWork[:IPADdress]:GATeway <Gateway>
Sets the IP address of the default gateway.

Parameters:

<Gateway>

string

Range:

0.0.0.0  to  ff.ff.ff.ff

Example:

SYSTem:COMMunicate:NETWork:IPADdress:GATeway

'1.2.3.4'

sets the IP address of the default gateway.

Manual operation:

See "Default Gateway" on page 107
