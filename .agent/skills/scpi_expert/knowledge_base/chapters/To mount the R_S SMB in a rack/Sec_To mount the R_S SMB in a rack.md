# To mount the R&S SMB in a rack
Use an adapter kit that fits the dimensions of the R&S SMB to prepare the instrument for rack mounting.

Order the rack adapter kit designed for the R&S SMB. For the order number, see the data sheet.

Mount the adapter kit. Follow the assembly instructions provided with the adapter kit.

Lift the R&S SMB to shelf height.

Push the R&S SMB onto the shelf until the rack brackets fit closely to the rack.

Tighten all screws at the rack brackets with a tightening torque of 1.2 Nm to secure the R&S SMB in the rack.
