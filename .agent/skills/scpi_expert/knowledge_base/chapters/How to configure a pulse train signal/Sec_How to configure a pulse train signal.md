# How to configure a pulse train signal
To perform pulse train generation, perform the following steps:

In the block diagram, select "Modulation > config... > Pulse Modulation".

Select "Source > Pulse Generator".

Select "Pulse Mode > Train".

![Picture](#/pictures/287)

The instrument displays the parameters required for configuring pulse train data.

Select "Pulse Train Data... > New List / Select List or File Manager".
