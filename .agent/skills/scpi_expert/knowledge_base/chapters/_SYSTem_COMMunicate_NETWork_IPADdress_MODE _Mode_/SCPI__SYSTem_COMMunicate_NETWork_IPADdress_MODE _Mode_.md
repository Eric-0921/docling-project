# :SYSTem:COMMunicate:NETWork:IPADdress:MODE <Mode>
Selects manual or automatic setting of the IP address.

Parameters:

<Mode>

AUTO | STATic

*RST:

n.a. (factory preset: AUTO)

Example:

SYSTem:COMMunicate:NETWork:IPADdress:MODE AUTO

The IP address is assigned automatically (DHCP)
