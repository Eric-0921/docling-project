# STANdard
An LF (Line Feed) is recognized as the terminator regardless of whether it is sent with or without EOI.

*RST:

n.a. (factory preset: STANdard)

Example:

SYSTem:COMMunicate:GPIB:LTERminator EOI

Only a character which is sent simultaneously with the line mes- sage EOI is accepted as the terminator.
