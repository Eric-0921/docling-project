# :SYSTem:COMMunicate:NETWork[:COMMon]:DOMain <Domain>
Sets the primary suffix, that is the DNS name without the host name part.


## Parameters:
<Domain>

string

Example:

SYSTem:COMMunicate:NETWork:COMMon:DOMain

'ABC.DE'

sets the domain of the network.

Manual operation:

See "DNS Suffix" on page 107

SYSTem Subsystem

Example:

SYSTem:SECurity:VOLMode:STATe "123456", 1 SYSTem:REBoot

Manual operation:

See "Volatile Mode" on page 119
