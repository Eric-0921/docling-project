# Setting up a VNC connection
Connect the instrument and the remote computer to a LAN, see Chapter 3.1.7, "Connecting to LAN", on page 24 .

Install the Ultr@VNC application and enable it on the instrument.

In the ICF firewall, enable communication on the network via Ultr@VNC program.

Install the VNC Viewer on the remote computer with Windows operating system, see "Installing the VNC Viewer on a Windows PC" on page 37.

Set up the VNC connection between the instrument and:

the remote computer with Linux/Unix operating system, see "Setting up the VNC connection on the Linux/Unix remote computer" on page 37;

the remote computer with Windows operating system, see "Setting up the VNC connection on the Windows remote computer" on page 38.
