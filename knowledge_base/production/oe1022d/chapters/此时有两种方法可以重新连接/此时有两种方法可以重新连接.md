注意 : 当 PC 机使用 USB 线连接 OE1022D 时，可以在电脑的设备管理器里确定当前的连 接使用的是哪个 COM 口，具体操作步骤是：选择 ' 我的电脑 '-> 右键 -> 属性 -> 设备管理器 -> 查 看 ' 端口 (COM 和 LPT)' 。
