## 4.6 [CHANNEL OUTPUT]子菜单

在前面板的 MENU 菜单栏选择 [CHANNEL OUTPUT] 子菜单进入，如图 31 所示：


<!-- image -->

[CHANNEL OUTPUT] 子菜单控制后面板上的两个 BNC 通道 CH1 和 CH2 输出用户需要的 R 、 X 、 Y 和 θ 值等，以及通过 <Offset&Expand> 设置输出的偏置与放大倍数。同时在 <Speed> 选项 栏设置通道输出数据速率。

输出信号的计算公式如下

\# \hat { B } \sqcup \L = \left ( \frac { \text {Signal} ( \# \tt t + \# \tt t \equiv \sqcup \tt t ) } { \text {Sens} } + \text {Offset} \right ) \times \text {Expand} \times 1 0 V

\hat { \# B J } \pm \frac { \text {Signal} ( \mathbb { J } \pm \hat { \# B J } \pm \hat { \# B F } \pm \hat { \# B F } ) } { 1 8 0 ^ { \circ } } \times 1 0 V

频率每个阶梯分 5 V-10 V

1000Hz = 5 V

1200Hz = 6 V

1600Hz = 8 V

1800Hz = 9 V

1990Hz = 9.95 V

2000Hz = 5 V （下一阶梯）
