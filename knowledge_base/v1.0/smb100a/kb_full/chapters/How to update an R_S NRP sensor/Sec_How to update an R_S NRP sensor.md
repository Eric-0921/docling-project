# How to update an R&S NRP sensor
To perform an R&S NRP sensor update proceed as follows:

Open the R&S website http://www.rohde-schwarz.com in section "Power Meters & Voltmeters > R&S NRP Sensors".

Select the respective sensor, e.g. R&S NRP18SN.

Select "Downloads > Firmware" and the offered firmware suitable for your sensor.

Transfer and save the firmware on the instruments, for example in the /var/user/ directory.

Connect the sensor to the R&S SMB and select "Setup > System > NRP Info Update" to open the dialog.

Select the sensor in the left sensor selection field.
