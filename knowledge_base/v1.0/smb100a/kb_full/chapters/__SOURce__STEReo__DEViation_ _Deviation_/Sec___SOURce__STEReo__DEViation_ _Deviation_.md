# [:SOURce]:STEReo[:DEViation] <Deviation>
Sets the MPX (Multiplex stereo signal) deviation.

Parameters:

<Deviation>

integer

Range:

0  to  depends on instrument hardware

*RST:

40 kHz

Example:

STER 40kHz

sets the stereo deviation according to standard.

Options:

R&S SMB-B5

Manual operation:

See "FM Deviation - Stereo Modulation" on page 218
