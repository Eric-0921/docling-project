# :DIAGnostic:INFO:OTIMe?
The command queries the number of operation hours.


## Return values:
<OTIMe>

float

Example:

DIAG:INFO:OTIM

queries the operation hours.

Response: 100023

The instrument was operated for 100023 hours up to now.

Usage:

Query only

Manual operation:

See "Operation Time / h" on page 98
