# :SYSTem:SAV <Pathname>
Saves the current R&S SMB settings into a file with defined filename and into a specified directory. The file extension ( *.savrcltxt ) is assigned automatically.


## Setting parameters:
<Pathname>

string

Example:

SYSTem:SAV "/var/user/temp/Test"

Saves the file Test.savrcltxt into the direc- tory /var/user/temp/ .

Usage:

Setting only

Manual operation:

See "Save" on page 129
