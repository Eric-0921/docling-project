# Address Mode
Selects the mode for assigning the IP address.

"Auto (DHCP)"

Assigns the IP address automatically, provided the network supports DHCP ( D ynamic H ost C onfiguration P rotocol)

The network used must support automatic assignment of the IP address via DHCP or APIPA (Zeroconf) in order to use this function.

"Static"

Enables you to assign the IP address manually.
