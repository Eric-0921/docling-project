# :DISPlay:PSAVe:HOLDoff <HoldoffTimeMin>
Sets the waiting time for the screen-save mode of the display.


## Parameters:
<HoldoffTimeMin>

integer

Range:

1  to  60

*RST:

n.a. (factory preset: 10)

Default unit: minute

Example:

DISP:PSAV:HOLD 8

Sets the timeout of the screen saver to 8 minutes.

Manual operation:

See "Wait Time" on page 108
