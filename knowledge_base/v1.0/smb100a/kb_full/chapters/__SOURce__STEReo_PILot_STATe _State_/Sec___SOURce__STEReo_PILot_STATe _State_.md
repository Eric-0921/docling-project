# [:SOURce]:STEReo:PILot:STATe <State>
Activates/deactivates the pilot tone generation.

Parameters:

<State>

0 | 1 | OFF | ON

*RST:

OFF

Example:

STER:PIL:STAT ON

activates generation of the pilot tone.

Options:

R&S SMB-B5

Manual operation:

See "Pilot State - Stereo Modulation" on page 220
