# :OUTPut<hw>:ALC:SEARch:MODE <Mode>
Activates/deactivates the RF output during the power search.

Parameters:

<Mode>

NORMal | MINimum

*RST:

NORMal

Example:

POW:ALC:SEAR:MODE NORM

during the power search, the RF output is active.

See "RF During Power Search - ALC" on page 155
