# :SYSTem:STARtup:COMPlete?
Queries if the startup of the instrument is completed.


## Return values:
<Complete>

0 | 1 | OFF | ON

*RST:

0

Example:

SYST:STAR:COMP?

// 1

// the startup of the instrument is completed

Usage:

Query only
