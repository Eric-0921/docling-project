# Parameters:

Parameters:

<Stop>

float

Determines the last level value of the sweep setting range.

Range:

full specified level range

Increment:

see the data sheet: Level sweep > Step size setting

resolution

*RST:

-10

Example:

POW:STOP 3

sets the stop level for the level sweep to 3 dBm for RF output A.

Manual operation:

See "Stop Level - Level Sweep" on page 191
