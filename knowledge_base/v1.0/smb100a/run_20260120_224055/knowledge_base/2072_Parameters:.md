# Parameters:

Parameters:

<OResponse>

string

Example:

SYSTEM:IDENT USER
        // Selects a user-defined identification
        SYSTEM:ORES "Test Option"
        // Defines the OPT string 'test option'
        *OPT?
        // Response: 'test option'

See "OPT String" on page 112
