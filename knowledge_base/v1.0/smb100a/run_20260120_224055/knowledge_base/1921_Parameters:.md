# Parameters:

Parameters:

<Stop>

float

Range:

-3  to  3

Increment:

1E-3

*RST:

3

Example:

SWE:LFC ON

activates the output of a linear voltage ramp from sweep start to sweep stop at the LF connector.

SWE:OVOL:STAR 0V

SWE:OVOL:STOP 3V

the voltage at the sweep start frequency is 0 V and at the stop frequency 3 V.

Manual operation:

See "Output Voltage Stop - RF Frequency Sweep" on page 187
