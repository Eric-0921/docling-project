# Setting_a_sweep

Setting a sweep

A sweep is set in five basic steps which are shown below taking a frequency sweep as an example.

The LF sweep is activated and configured in the "Mod Gen" block.

Set the sweep range ("Start Freq" and "Stop Freq" or "Center Freq" and "Span").

Select linear or logarithmic sweep spacing ("Spacing").

Set the step width ("Step Lin/Log") and dwell time ("Dwell Time").

Activate the sweep ("Mode" to Auto, Single, Step or Extern Single, Extern Step).
