# Mode

Mode

Selects the way the instrument identification is performed.

"Automatic"

Sets the "IDN String" and the "OPT String" automatically for the instrument selected with the parameter Language.

"User Defined"

Enables you to define the "IDN String" and the "OPT String" for the instrument selected with the parameter Language.
