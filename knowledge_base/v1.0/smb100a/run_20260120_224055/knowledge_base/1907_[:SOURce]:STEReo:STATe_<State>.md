# [:SOURce]:STEReo:STATe_<State>

[:SOURce]:STEReo:STATe <State>

Activates/deactivates stereo modulation.

Parameters:

<State>

0 | 1 | OFF | ON

*RST:

OFF

Example:

STER:STAT ON

activates generation of stereo signal.

Options:

R&S SMB-B5

Manual operation:

See "State - Stereo Modulation " on page 218
