# Parameters:

Parameters:

<Frequency>

float

Range:

0.1  to  1E6

Increment:

0.01

*RST:

1000

Example:

STER:SOUR LFG

The internal LF generator is used as modulation source for the audio signal.

STER:AUD:FREQ 1100

sets the frequency of the audio signal to 1.1 kHz

Options:

R&S SMB-B5

Manual operation:

See "LF Gen Freq - Stereo Modulation" on page 219
