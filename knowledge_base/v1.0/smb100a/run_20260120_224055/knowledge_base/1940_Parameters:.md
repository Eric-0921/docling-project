# Parameters:

Parameters:

<Dwell>

float

Range:

1E-3  to  100

Increment:

100E-6

*RST:

15E-3

Example:

SWE:POW:DWEL 12 ms

sets a dwell time of 12 ms for a level sweep at the RF output.

Manual operation:

See "Dwell Time - Level Sweep" on page 192
