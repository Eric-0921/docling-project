# Example:

Example:

SOUR:LFO:SWE:FREQ:MODE AUTO

TRIG0:SWE:SOUR SING

SOUR:LFO:FREQ:MODE SWE

SOUR:LFO:SWE:FREQ:EXEC

Generates the sweep signal step-by-step, manually triggered. To perform the sweep steps, enter the frequency value under Current Freq.
