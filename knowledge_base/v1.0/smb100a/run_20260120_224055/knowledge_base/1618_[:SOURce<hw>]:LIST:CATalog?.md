# [:SOURce<hw>]:LIST:CATalog?

[:SOURce<hw>]:LIST:CATalog?

Requests a list of available lists. The individual lists are separated by commas.

The lists are stored with the fixed file extensions *.lsw in a directory of the user's choice. The directory applicable to the commands is defined with the command MMEMory:CDIR .
