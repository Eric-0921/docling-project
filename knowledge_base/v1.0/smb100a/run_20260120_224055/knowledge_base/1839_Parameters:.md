# Parameters:

Parameters:

<Width>

float

Range:

10 ns  to  100 s

Increment:

10 ns

*RST:

2 us

Example:

PULM:WIDT 33 us

sets a width of 33 us for the pulse.

Options:

R&S SMB-K23 (Pulse Generator)

Manual operation:

See "Pulse Width - Pulse Generator" on page 233
