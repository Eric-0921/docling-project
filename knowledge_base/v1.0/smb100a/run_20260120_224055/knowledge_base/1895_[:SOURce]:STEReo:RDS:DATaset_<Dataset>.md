# [:SOURce]:STEReo:RDS:DATaset_<Dataset>

[:SOURce]:STEReo:RDS:DATaset <Dataset>

Selects one of the five data sets provided on the instrument for use in the RDS signal.

Operating Manual 1407.0806.32 ─ 23

418

R&S ® SMB100A

Remote Control Commands

Parameters:

<Dataset>

DS1 | DS2 | DS3 | DS4 | DS5

*RST:

DS1

Example:

STER:RDS:DAT DS5

activates use of data set 5 for generation of the RDS signal.

Options:

R&S SMB-B5

Manual operation:

See "RDS Data Set - Stereo Modulation" on page 222
