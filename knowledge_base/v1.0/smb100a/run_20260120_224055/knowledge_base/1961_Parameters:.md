# Parameters:

Parameters:

<Logarithmic>

float

Increment:

0.01

*RST:

1

Example:

SWE:POW:STEP 10dB

sets the step width for logarithmic sweep spacing to 10 dB of the previous level in each instance (for a level sweep).

Manual operation:

See "Step - Level Sweep" on page 192
