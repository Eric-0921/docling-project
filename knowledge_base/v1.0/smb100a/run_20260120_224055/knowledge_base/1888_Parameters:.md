# Parameters:

Parameters:

<Impedance>

600 | 600Ohm | 100000 | 100kOhm | 100000Ohm

*RST:

100000

Example:

SOUR:STER:EXT:IMP 600Ohm

selects 600 OHM as the impedance for the external analog audio signals.
