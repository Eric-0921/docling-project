# LOGarithmic

LOGarithmic

With the logarithmic sweep, the step width is a constant fraction of the current frequency. This fraction is added to the current fre- quency. The logarithmic step width is entered in % (see [:

SOURce<hw>]:LFOutput:SWEep[:FREQuency]:STEP: LOGarithmic on page 362).
