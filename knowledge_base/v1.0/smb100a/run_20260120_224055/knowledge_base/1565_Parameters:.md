# Parameters:

Parameters:

<Mode>

DECimal | USER

*RST:

DECimal

Example:

FREQ:STEP 50 kHz

sets the step width for the frequency setting to 50 kHz. FREQ:STEP:MODE USER

actives this step width for frequency variation with the rotary knob (manual control) and with frequency values UP/DOWN (remote control).

See "Variation Active" on page 141
