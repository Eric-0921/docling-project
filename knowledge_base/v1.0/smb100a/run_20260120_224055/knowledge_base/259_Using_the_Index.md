# Using_the_Index

Using the Index

Select "Index".

Enter the first characters of the topic you are interested in. The entries starting with these characters are displayed.

Press the [ENTER] key to change the focus.

Use the [Up/Down] keys to navigate and select the suitable keyword.

Press the [ENTER] key to display the help topic.

The corresponding help topic is displayed.
