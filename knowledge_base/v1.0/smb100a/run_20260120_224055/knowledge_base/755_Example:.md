# Example:

Example:

SOUR:SWE:POW:MODE AUTO

TRIG:PSW:SOUR EXT

SOUR:POW:MODE SWE (External trigger)

Sets a step-by-step sweep cycle. Each sweep step is triggered by an external trigger signal (trigger source as described under "Extern Single"). The step width corresponds to the step width of the rotary knob.
