# Navigating_in_the_help_topics

Navigating in the help topics

To scroll through a page, use the [Up/Down] arrow keys.

To follow a cross-reference, select the link text.

To return to the previous page, select "Back". This function scrolls back all steps that you have performed before.

Use the "Scroll Right" or "Scroll Left" buttons to shift the indicated area of the navigation window to the left or right.
