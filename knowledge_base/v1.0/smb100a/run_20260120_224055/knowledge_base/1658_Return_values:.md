# Return_values:

Return values:

<Points>

integer

Range:

0  to  INT_MAX

*RST:

0

Example:

LIST:SEL '/var/list3'

selects list3 for editing. The R&S SMB generates a new file automatically, if it does not exist yet.

LIST:POW:POIN?

queries the number of levels in the list file

Response:

327

Usage:

Query only
