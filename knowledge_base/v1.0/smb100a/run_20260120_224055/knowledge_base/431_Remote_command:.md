# Remote_command:

Remote command:

:HCOPy:FILE[:NAME]:AUTO:DIRectory:CLEar on page 302

"Prefix, Year,

Month, Day"

Determines the rules for "Automatic Naming".

Per default, the automatically generated file name is composed of: <Path>/<Prefix><YYYY><MM><DD><Number>.<Format> , where Y , M and D mean Year, Month, Day; Number is the "Current Auto Number".

You can deactivate/activate each component separately. The "Resulting File Name" indicates the current file name syntax.
