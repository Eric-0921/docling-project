# Parameters:

Parameters:

<Mode>

IMPort | EXPort

*RST:

IMPort

Example:

LIST:DEXC:MODE IMP

selects that ASCII files with frequency and level value pairs are imported and transferred into list mode lists.

LIST:DEXC:AFIL:SEL '/var/list.csv'

selects that ASCII file list.csv is imported.

LIST:DEXC:SEL '/var/list_imp'

selects that the ASCII file list.csv is imported into list mode

list list_imp .

Manual operation:

See "Mode - List Mode" on page 199
