# Parameters:

Parameters:

<Mode>

DECimal | USER

*RST:

DECimal

Example:

POW:STEP 2

sets the step width for the level setting to 2 dB.

POW:STEP:MODE USER

actives this step width for level variation with the rotary knob (manual control) and with level values UP/DOWN (remote con- trol).

Manual operation:

See "Variation Active" on page 152
