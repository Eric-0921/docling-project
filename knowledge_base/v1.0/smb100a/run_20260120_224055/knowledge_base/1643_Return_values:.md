# Return_values:

Return values:

<Points>

integer

Range:

0  to  INT_MAX

*RST:

0

Example:

LIST:SEL '/var/list3'

selects list3 for editing. The R&S SMB creates a new file automatically, if it does not exist yet.

LIST:FREQ:POIN?

queries the number of frequency values in the list Response: 327

Usage:

Query only
