# Parameters:

Parameters:

<Dwell>

float

Range:

2E-3  to  100

Increment:

100E-6

*RST:

15E-3

Example:

SWE:DWEL 12 ms

sets a dwell time of 12 ms for a frequency sweep at the RF out- put.

Manual operation:

See "Dwell Time - Frequency Sweep" on page 186
