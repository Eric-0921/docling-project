# Parameters:

Parameters:

<MTime>

float

Range:

1  to  999.99

Increment:

0.01

*RST:

4

Default unit: s

Example:

SENS1:FILT:TYPE NSR

selects fixed noise filter mode for the power sensor connected to the SENSOR connector.

SENS1:FILT:NSR .2

sets a noise content of 0.2.

SENS1:FILT:NSR:MTIM 5

limits the settling time to 5 seconds

Manual operation:

See "Timeout" on page 177
