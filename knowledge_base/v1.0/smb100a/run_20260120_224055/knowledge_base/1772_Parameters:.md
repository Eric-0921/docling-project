# Parameters:

Parameters:

<Width>

float

Range:

10 ns  to  100 s

Increment:

10 ns

*RST:

3 us

Example:

PULM:DOUB:WIDT 33 us

sets a width of 33 us for the second pulse.

Options:

R&S SMB-K23 (Pulse Generator)

Manual operation:

See "Double Pulse Width - Pulse Generator" on page 233
