# Changing_the_GPIB_address_of_the_instrument

Changing the GPIB address of the instrument

The GPIB address can be changed manually or using a remote control command.

Manually: press the [SETUP] key.

Select "Remote > GPIB".

Enter the GPIB address.

Remotely: use the remote control command: SYST:COMM:GPIB:ADDR 18
