# Setting_parameters:

Setting parameters:

<State>

0 | 1

Example:

STEReo:DIRect "TP=1"

The traffic program flag is set to "1".

Example:

STEReo:DIRect? "TP"

Response: "1"

STEReo:DIRect "TRANS<DataNumber>=<DataStream>" STEReo:DIRect? "TRANS<DataNumber>"

Transparent mode.

An RDS data stream of binary data is generated. If transparent data is selected, all other RDS data is ignored.
