# SYSTem_Subsystem

SYSTem Subsystem

Example:                SYST:IDENT USER
                                    // Selects a user-defined identification
                                    SYST:IRES "Test Device"
                                    // Defines identification string 'test device'
                                    *IDN?
                                    // Response: 'test device'
