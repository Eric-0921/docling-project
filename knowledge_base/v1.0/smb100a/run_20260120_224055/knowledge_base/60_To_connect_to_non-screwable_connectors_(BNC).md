# To_connect_to_non-screwable_connectors_(BNC)

To connect to non-screwable connectors (BNC)

Use a high-quality RF cable that matches the RF connector type. See "Cable selection and electromagnetic interference (EMI)" on page 23.

To connect the RF cable with the RF connector, proceed as follows:

Carefully align the connector of the cable and the RF connector along a common axis.
