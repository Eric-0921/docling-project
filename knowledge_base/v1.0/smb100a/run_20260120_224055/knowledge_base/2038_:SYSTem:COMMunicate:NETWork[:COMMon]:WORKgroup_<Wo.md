# :SYSTem:COMMunicate:NETWork[:COMMon]:WORKgroup_<Wo

:SYSTem:COMMunicate:NETWork[:COMMon]:WORKgroup <Workgroup>

Sets the individual workgroup name of the instrument.

Parameters:

<Workgroup>

string

Example:

SYSTem:COMMunicate:NETWork:COMMon:WORKgroup

'TEST_09'

sets the workgroup name
