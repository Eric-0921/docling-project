# Double_Pulse_Delay_-_Pulse_Generator

Double Pulse Delay - Pulse Generator

(Double Pulse only)

Sets the delay from the start of the first pulse to the start of the second pulse.

Remote command:

[:SOURce<hw>]:PULM:DOUBle:DELay on page 396
