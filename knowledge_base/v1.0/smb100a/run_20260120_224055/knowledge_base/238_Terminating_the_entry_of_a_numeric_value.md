# Terminating_the_entry_of_a_numeric_value

Terminating the entry of a numeric value

To terminate the entry of a numeric value:

Press the rotary knob (= Enter).

Press a [UNIT] key on the front panel.

Select a "Unit" in the selection field next to the parameter value.
