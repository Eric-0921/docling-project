# Navigating_in_the_table_of_contents

Navigating in the table of contents

To navigate within the table of contents entries, use the [Up/Down] keys. Entries that contain further entries are marked with a plus sign.

To display a help topic, press the "ENTER" key.

The corresponding help topic is displayed.
