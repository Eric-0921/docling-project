# STEP

STEP

Each trigger triggers one sweep step only. The frequency increases by the value entered with [:SOURce<hw>]:LFOutput:

SWEep[:FREQuency]:STEP[:LINear] or [:SOURce<hw>]: LFOutput:SWEep[:FREQuency]:STEP:LOGarithmic .

*RST:

AUTO

Example:

LFO:SWE:MODE AUTO

selects Auto mode.

Manual operation:

See "Mode" on page 226
