# Parameters:

Parameters:

<Start>

float

Determines the first level value of the sweep setting range.

Range:

full specified level range

Increment:

see the data sheet: Level sweep > Step size setting

resolution

*RST:

-30

Example:

POW:STAR -20 dBm

sets the start level for the level sweep to -15 dBm for RF output A.

Manual operation:

See "Start Level - Level Sweep" on page 190
