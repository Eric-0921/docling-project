# EXCLude

EXCLude

Retains the current level when an instrument configuration is loaded.

*RST:

INCLude

Example:

POW:RCL INCL

takes the level value from an instrument configuration loaded with command *RCL .

Manual operation:

See "Exclude Level" on page 131
