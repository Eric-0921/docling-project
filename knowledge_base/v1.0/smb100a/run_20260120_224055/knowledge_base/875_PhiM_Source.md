# PhiM_Source

PhiM Source

Selects the source for the PhiM signal.

"Internal"

Uses the internal LF generator as the modulation signal source for PhiM.

"External"

Uses an externally applied modulation signal.

The external signal is input via the [MOD EXT] connector.

"Internal + External"

Uses both, the internal and externally applied modulation signal.
