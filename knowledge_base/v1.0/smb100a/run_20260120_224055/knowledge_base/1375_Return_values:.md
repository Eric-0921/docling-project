# Return_values:

Return values:

<Lower>

float

Increment:

0.01

Example:

OUTP:AFIX:RANG:LOW?

queries the minimum level for the FIXed setting.

Example:

Response: -50

The minimum level is -50 dBm.

Usage:

Query only

Manual operation:

See "Fixed Range (PEP) In" on page 151
