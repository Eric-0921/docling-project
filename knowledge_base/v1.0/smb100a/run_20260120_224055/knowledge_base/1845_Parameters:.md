# Parameters:

Parameters:

<State>

0 | 1 | OFF | ON

*RST:

n.a. (factory preset: 0)

Example:

ROSC:SOUR EXT

Selects the external source. The reference must be input at the REF IN input.

Example:

ROSC:EXT:RFOF:STAT ON

If the external signal is missing, no RF signal is output.
