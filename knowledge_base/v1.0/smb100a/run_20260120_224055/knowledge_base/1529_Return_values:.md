# Return_values:

Return values:

<Sensitivity>

float

Range:

0  to  max

Increment:

0.01

Example:

FM:DEV 5E3

sets a modulation deviation of 5 kHz.

FM:SENS

queries the input sensitivity at the external modulation input.

Response: 5E3

since the voltage value for full modulation is 1V, the resulting sensitivity is precisely 5000 Hz/V.

Usage:

Query only

Manual operation:

See "FM Sensitivity" on page 210
